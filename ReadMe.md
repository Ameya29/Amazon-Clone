
# Amazon Clone

A static clone of the popular e-commerce website Amazon built using HTML & CSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/Ameya29/Amazon-Clone
```

Go to the project directory

```bash
  cd Amazon-Clone
```

Go live from the HTML file
